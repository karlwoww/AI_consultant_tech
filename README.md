# AI Consultant Tech

AI-ассистент магазина электроники — виртуальный консультант, работающий строго на основе предоставленных данных.

## О проекте

Ассистент помогает пользователям получать информацию о товарах, магазинах, доставке и гарантии без участия живого оператора.

Проект создан для практики разработки AI-ассистентов и обработки пользовательских запросов с использованием технологий искусственного интеллекта.

---

## Возможности

- ответы на вопросы о наличии, ценах и характеристиках товаров;
- информация об адресах и времени работы магазинов;
- данные о гарантии и сроках её действия;
- стоимость и сроки доставки;
- контактная информация службы поддержки.

---

## Ключевые особенности

### Точность
Используются только подтверждённые данные без генерации вымышленных ответов.

### Структурированность
Ассистент сохраняет форматирование ответов: списки, абзацы и разделение информации.

### Прозрачность
Корректно обрабатываются многосоставные вопросы.

### Надёжность
Если информация отсутствует, система сообщает:
> «Информация не найдена»

---

## Принцип работы

1. Анализ пользовательского запроса;
2. Определение категории вопроса;
3. Поиск информации в базе знаний;
4. Формирование ответа пользователю.
